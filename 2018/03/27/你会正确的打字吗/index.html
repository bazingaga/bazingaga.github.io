你会正确的打字吗

发表于 2018-03-27 | 更新于 2018-04-03 | 分类于 平淡生活
字数统计 1,428 字 | 阅读时长 5 分钟

你有坚持很久的习惯吗，比如手机 APP 主屏的摆法、跑步的姿势等。这些习惯可能会让你在平常的学习工作中得心应手，但当你想要更高效地完成任务时，这些旧时的习惯可能会阻碍你。

比如游泳，当你发现别人踢水一次滑行的距离比你踢两次都要远很多，而且并不费力，你就应该考虑自己的姿势是不是不标准，不科学了。

比如我最近就在矫正自己键盘打字的指法。

一、规范的重要性

我们在学习一项新的技能时，身体因为陌生而不适应。所以我们的动作笨拙缓慢。但随着肌肉慢慢熟悉这些动作，我们就会变得越来越熟练，也越来越游刃有余。

在这个学习过程中的技巧诀窍随着前人的沉淀，逐渐变为某些标准，这些标准是每个新手都要遵循的规则，这并不是简单的循规蹈矩，而是其中包含着前人的智慧和经验会帮助我们少走弯路，并在后期有更大的提升空间。

二、不规范键位的弊端——每个手指分配的键位不均等

  • 打字容易疲劳

    不标准打字的姿势会导致打字效率很低，因为不科学的指法不能把打字频率合理的分配到每个手指上。对于我来说，打字常用的手指是食指和中指，无名指和小指的使用场景不大，以至于很容易累。

  • 打字速度有瓶颈

    经常使用自己熟悉的手指就会导致少数的手指需要覆盖键盘上大部分区域，手的移动范围大了，很容易累，对速度也有抑制。

  • 不能盲打

    常用手指负责的范围过大，肌肉难记住所有的键位，需要一边看键盘，一边看屏幕，也制约了速度。

所以练习使用正确的键位十分重要。

三、正确的键位

1. 字母键

正确的键位分布：

简单来说，就是通过减少手指的移动，来提升打字速度和缓解疲惫感。

不过这个图并不是十全十美，比如右小指负责的范围还是过大了，我们需要给小指「减负」。

王垠提到：

  1. 用无名指去打第一排的 0, -, =, delete。
  2. 不要用右边的 ctrl 和 alt 键，只用左边的。
  3. 使用左手拇指去按左边的 ctrl，alt 和 command。

扩大右手中指的负责范围是个不错的解决方案。

2. 数字键

中文打字还常常需要用到数字键选字，但常用的也只是前三个或前四个。

  • 1=小指
  • 2=无名指
  • 3=中指

3. 大写键

大写键的原则就是尽量将左右手合理分工，即右手按字母键的时候，左手去按大写键，尽量不要一只手又按字母键又按大写键。

四、如何矫正错误的指法

1. 使用键位矫正打字网站

我推荐 typingclub，这也是我在用的练习网站。网站本身针对英文打字，一开始可以先用熟悉字母键位。

网站将近 300 多关卡，关卡设计遵循循序渐进的原则，一开始针对几个特定的键位，到后边也会有综合训练，其中还会穿插小游戏和讲一些「打字」的延伸知识。能做到很好的劳逸结合。

其实对于打字来说，英文输入的效率要高于中文。

2. 低空飞行，放慢速度

键位记忆需要几天的时间来适应，在这个过程中，你或许会感到很别扭，总想用以前自己熟悉的打字方式，这个时候就需要你慢下来，手尽量「低空飞行」，也就是尽量控制手移动的幅度，将手尽可能的「粘」在键盘上，这样一开始会比较别扭，但对于矫正键位来说，十分有帮助。

3. 长期坚持

短时间内，你以前的打字习惯还是会在你的指尖重现，包括我在写这篇文章时，还是需要刻意控制指法。不过在坚持一段时间之后，等新的肌肉记忆形成，就会好很多。

五、你真的需要很快的输入速度吗

练习规范键位，很大的原因是我们想追求更快的键盘输入速度，但是我们真的需要特别快的输入速度吗。如果你是一个打字员文秘这样的职位，你当然需要很快的打字速度。但是你如果只是想写写文章，打打代码，过快的输入速度对我们的帮助其实并不是很大，因为在写文章和码代码的大部分情景，都是一个缕清思路的过程，这个过程往往并不需要打字很快，你应该更专注于你的思路上，文章的结构上。思考要比敲击键盘重要，况且思考的过程会经常停顿，并不是完全一气呵成。这些停顿多是在做一个总结回顾与思考。

打字的过程本身就是一个权衡打字速度和准确度的过程。对于提升速度来说，不妨更注重准确度的练习，打的飞快但错误频出，经常要按退格键，这样还不如慢些打，保证准确度。
