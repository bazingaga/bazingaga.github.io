「辞职」和「买房」的问题是共通的

发表于 2018-05-03 | 更新于 2018-05-03 | 分类于 平淡生活
字数统计 1,309 字 | 阅读时长 4 分钟

今天，又有人来问我「想辞职，不知道自己应该学什么」。很遗憾，我并不能给你什么建设性的建议，因为我自己都还没有走出困境，又如何有资格指点别人的选择呢。

我特别害怕别人问我一个问题：「你天天都在做什么？」说实话，这个问题我是最难回答的，这也是个最触及我这段时间真实面貌的问题，我不愿回答。

我一直想尽快走入生活的正轨，可我不知道为什么就这么难。我总是无法和过去的生活告别，我总是沉浸在过去的痛苦与失败中，幻想从前错误的选择，不得体的做法会对现在及以后的我产生影响。

今天又谈到了买房的话题，或许尽快买房在当下是明确的选择，但是我总是很犹豫。

我说过自己是一个困惑与矛盾的人，有一个东西我明明很想要，但我就惧怕获得这个东西所付出的代价自己不能承受。

明明可以买一个可以报销的「一拖多」这样就不用在我接 Kindle 的时候，拔掉键盘或鼠标的线，也可以弥补外接移动电源接口不够，电力不足的缺憾。但我仍然很难抉择，我好像对于自己要花的钱十分苛刻，时刻要求它们给我最大的回报，一分钱都要讲究性价比。

所以，我迟迟买不了房。

有的人适合留在北上广，有的人不适合，因缘巧合，我应该属于不适合留下来的那一批里面，因为我不想累，不想吃苦，不想疲于人际交往，不想努力，不想坚持做完一件事情。

由于敬礼的缘故，我有买房资格，这对别人来说，能够绕过「五年限购」真的非常不错，因为你想不到五年后的房价是一个什么情况。

如果在「适不适合」这方面延伸来讲，我无疑又是非常适合的，买房资格，小户型，低房贷，而且不止我一个人还贷款。我想不出我为什么不买房。

而我对现在的租房生活又是不满意的。

租房到底对我有什么影响，我觉得最大还是无奈。

  • 住在郊区，市区有活动，时间但凡晚一点就赶不上末班地铁。
  • 合租，害怕尴尬，所以我不喜欢和人打照面，我出门的时候要看别人不在公共区域才会出去。
  • 上厕所的时候，厕所总是有人。
  • 房子过小，摆放东西比较乱，也影响自己做事情的动力。
  • 敬礼不高兴
  • 没有任何提升生活质量的想法。

这是一种无奈，对现状的妥协，对改变现状的无奈。

你为什么宁愿每天都过不好的生活，也不愿改变？

我觉得这样的原因有两个：

  • 舒服
  • 告诉自己只是没努力，不是努力了不行，不参与竞争

都是比较鸵鸟的想法。尤其第二种，就像小时候的「不是我不学，只是我不想学而已」。

我觉得社交网络真是人类烦恼的最大源头，人们明明可以走自己的路，有的快，有的慢，但是社交网络出现之后，有钱人的生活让普通人熟知，我们也想过上物质充裕的生活，我们也想在社交媒体上展示自己好的一面，但人们慢慢的就变了，开始变得攀比，要面子。

我也不例外，我知道已经有很多人比我优秀，就算我现在玩命努力，但不会影响他们行走的脚步，所以很大程度上，我会永远追不上他们，至少在社交媒体上追不过。

我们都迷失了自己，不知道自己真正想要什么，自己追寻的东西到底是不是自己真正想要的。

我到底想要什么呢？或者我要追寻什么呢？

我说不准，但我知道的是，绝不仅仅是一套房子而已。

而且我害怕的是，这套房子之后，还有下一套，而且要考虑孩子，所以说我接下来很长的时间都要和房子交战了？

还是说房子是根，只有先搞定房子才能去追寻那些自己真正喜欢的东西，但那个时候，我还有时间，还有精力吗？

我写东西有不好的习惯，就是很少使用疑问句，我总想让很多东西直接下定论，但是我明明知道，我的定论很多时候是不准的，更何况，有的问题本来就没有答案。

所以我决定以后，尽量多使用疑问句。能提出问题总比给出错误的答案要强。

我也很迷茫，所以我真的给不了你很多建议，而且，自己的问题只有自己才能解决，别人真的无能为力。
