开始学习vue，遇到了一些问题

发表于 2017-03-23 | 更新于 2018-01-28
本文字数： 734 | 阅读时长 ≈ 1 分钟

最近看了很多的招聘要求，发现很多都需要有掌握框架的要求，vue作为一个华人独立开发的框架，能够与大厂出品的react，angular相提并论，也说明是很不容易的。最近webstorm也开始支持vue的语法高亮了，可见vue在全世界范围内的影响力可见一斑。

安装和使用的时候就遇到了一些问题

首先按照官网命令行安装很顺利，直到这一步的时候


开始报错

网上查了查是node的版本过低，于是更新了node的版本，毕竟快一年都没更新node了。。

然而更新node时也出现了问题

网上查了查，原来自己以前是homebrew安装的node，可能先要卸载

然后再安装node，就可以了。

cd到项目目录，也可以成功运行服务器了

然而问题还没有完，hexo又出问题了

网上搜了rebuild也无济于事

再去搜索了一下，卸载然后重新安装hexo

这里很不爽，竟然报了个warning

网上查了查，大意就是说只是个warning，不用大惊小怪啦，外国小哥说话还是很萌的

重新安装hexo的时候还是建议大家使用国内的npm镜像，比如淘宝的，国情的原因，下载的速度还是不如国内。

万里长征第一步，遇到了一点问题，还是慢慢来好了，最近和同住的口罩创业大叔聊了聊，找第一份工作还是要严肃一点，不可以将就。慢慢找，一定要找一个自己喜欢的，并且互相可以共同发展的工作，最近开始重拾js高程的复习了，vue也开始并驾齐驱，可能是天气慢慢回暖，自己也有心情做这些事情了吧。

我觉得以后的生活还是要乐观面对的，不管现在如何，可能我这个年纪的年轻人，真的没有几个轻松的吧，如果想要过上自己希望的生活就不能太轻松。自己平时不放松，有一颗闲得下来的上进心，慢慢积累，应该就会有一些收获。
